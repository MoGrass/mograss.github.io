Github+jsdelivr+PicGo搭建超快速个人图床

设置Token

首先，新建一个仓库

不要勾选 “Private”。勾选了就不可以当图床了

然后去设置 Token，点你的”头像”(虽然我这里是显示不出来)，然后点击”Settings”

选择Developer settings.

接着选择Personal access tokens

新建一个Token

弄完以后你会得到一个Token，记得保存好！这个Token只出现一次！刷新了就没了

配置PicGo

打开PicGo官网下载一个最新的PicGo

安装完打开，设置一波。
先把Github的设置给弄下，要输入你的仓库，和分支(直接输入”Master”就ok了)，然后是Token.

然后把 Server 给关闭，不然的话上传不了图片

然后就可以了。

创建Release

不创建的话，自己享受Github加载图片的速度吧

然后就可以了

使用jsdelivr

jsdelivr是世界上唯一一个免费CDN。支持Github加速、NPM、WORDPRESS
